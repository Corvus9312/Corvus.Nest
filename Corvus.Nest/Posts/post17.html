Line bot-建立帳號

建立Line bot帳號，並取得Channel Access Token及User ID

1.進入Line Developer網站。

https://developers.line.biz/zh-hant/

2.點選Console，登入自己的Line帳密，進入主控台。

3.建立Provider。

若是第一次登入，需要填寫個人資訊。之後，系統會讓你建立一個新的Provider。 這邊可以Key公司名稱，或自己的名字、或代號都可以，並無限制。

4.建立Messaging API頻道。

Provider建立完畢後，點擊「Create a Messaging API Channel」建立頻道。接著填寫名稱、說明、分類、子分類及個人EMail，並勾選全部同意條款後，點擊「Create」新增頻道。

5.加入Line bot好友。

進入「Messaging API」，Bot information有Bot basic ID及QR code，這個就是這次建立Line bot的ID。 複製Bot basic ID去加入好友搜尋用戶ID，或直接掃描QR code來加入機器人好友就可以了。

6.取得Token。

首先Channel access token是控制Line bot的鑰匙，所以必須要保存好不能外流。 位置在「Channel settings」最底下的「Channel access token (long-lived)」，點擊「Issue」取得來取得Channel access token。

6.取得User ID。

接著是User ID，是你自己的ID。透過Line bot傳送訊息給自己時，就需要使用這個ID來發送訊息。 位置在「Basic settings」最底下的「Your user ID」，直接複製這串U開頭的英數字串就可以了。
